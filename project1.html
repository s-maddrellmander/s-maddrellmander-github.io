GNN for Molecular Prediction

As part of the competition-winning research team at the OGB Large Scale Challenge (NeurIPS), I developed GPS++, a novel Graph Neural Network model for molecular property prediction. This project showcased the power of combining message passing neural networks with transformers for improved performance on large-scale molecular datasets.

Key Achievements

  • Developed GPS++, a hybrid model combining GNNs and Transformers
  • Achieved state-of-the-art performance on the OGB-LSC PCQM4Mv2 benchmark
  • Implemented efficient scaling techniques for handling large molecular graphs
  • Contributed to the open-source Graphium library for GNN research
